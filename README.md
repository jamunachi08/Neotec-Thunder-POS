# Neotec Thunder POS (ERPNext v15+)

Install-safe Frappe Cloud app providing **configurable foundations** for:
- Restaurant POS (Tables/Floors/Sessions, Split Bills, KDS)
- Retail POS (offers, loyalty, barcode-ready structure)
- Grocery POS (weight items, offers, loyalty)
- Web ordering (settings + orders + menu categories) + docs pages

This package focuses on installation reliability and front-end configuration.
